# AcidDevTools

Trusty sidekick for [R][] package development.

## Installation

This is an [R][] package.

```r
install.packages(
    pkgs = "AcidDevTools",
    repos = c(
        "https://r.acidgenomics.com",
        getOption("repos")
    ),
    dependencies = TRUE
)
```

[r]: https://www.r-project.org/
